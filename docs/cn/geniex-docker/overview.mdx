## **Linux Docker 方案**

GenieX SDK Docker 使你能够在 Linux ARM64 系统的容器化环境中运行 AI 模型，针对高通 NPU 设备（如 Dragonwing IQ9）进行了优化。该方案提供一致、隔离的环境来运行推理工作负载，支持服务器模式（REST API）和交互式 CLI 使用。

## **核心特性**

- **容器化部署**：在隔离的 Docker 容器中运行 GenieX SDK
- **NPU 加速**：针对 Linux ARM64 上的高通 NPU（Dragonwing IQ9）优化
- **双模式**：支持服务器模式（REST API）和交互式 CLI
- **简单设置**：简单的拉取和运行命令
- **多种模型类型**：支持 LLM、嵌入、重排序、计算机视觉和 ASR 模型

## **支持的模型**

| 模态 | 模型 |

| **重排序** | [`NexaAI/jina-v2-rerank-npu`](https://huggingface.co/NexaAI/jina-v2-rerank-npu) |

| **ASR** | [`NexaAI/parakeet-tdt-0.6b-v3-npu`](https://huggingface.co/NexaAI/parakeet-tdt-0.6b-v3-npu) |
