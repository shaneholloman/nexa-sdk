## **快速概览**

### **安装**

```bash
pip install -i https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple geniex
```
这将从 [test PyPI](https://test.pypi.org/project/geniex/) 下载 Python 包。

<Note>
**高通内网用户：** 高通 VPN 访问 Hugging Face 存在连接问题。请先手动下载模型，然后将模型的**本地路径**传给 `from_pretrained`，而不是使用 HF 仓库 ID。
</Note>

### **LLM 推理**

以下是一个简单的入门示例：

```python
from geniex import AutoModelForCausalLM

model = AutoModelForCausalLM.from_pretrained(
    "qwen3",               # 简称别名、HF 仓库 ID 或本地 GGUF 路径
    device_map="auto",     # "auto" | "<plugin>" | "<plugin>:<device>"
)

messages = [{"role": "user", "content": "What is 2+2?"}]
prompt = model.tokenizer.apply_chat_template(
    messages, tokenize=False, add_generation_prompt=True,
)

# 一次性生成
output = model.generate(prompt, max_new_tokens=256)
print(output.text)
print(f"[{output.profile.generated_tokens} tok, "
      f"{output.profile.decode_speed:.1f} tok/s, stop={output.profile.stop_reason}]")

# 流式生成
streamer = model.generate(prompt, max_new_tokens=256, stream=True)
for chunk in streamer:
    print(chunk, end="", flush=True)

model.close()
```

### **VLM 推理**

首先，下载一张示例图片：

```bash
curl -o demo.jpg https://qaihub-public-assets.s3.us-west-2.amazonaws.com/qai-hub-geniex/demo.jpg
```

然后使用下载图片的完整路径运行推理：

```python
import os
from geniex import AutoModelForVision2Seq

image_path = os.path.abspath("demo.jpg")

model = AutoModelForVision2Seq.from_pretrained("qualcomm/Qwen2.5-VL-7B-Instruct", device_map="auto")
messages = [{
    "role": "user",
    "content": [
        {"type": "image", "image": image_path},
        {"type": "text", "text": "Describe the image."},
    ],
}]
prompt = model.tokenizer.apply_chat_template(
    messages, tokenize=False, add_generation_prompt=True,
)
print(model.generate(prompt, images=[image_path]).text)
```

### **模型管理**
CLI 使用的模型管理器同样可以通过代码调用：

```python
from geniex import model_manager as mm

mm.pull("qwen3")                      # 别名或 "org/repo[:quant]"
paths = mm.get_paths("qwen3")         # 解析本地路径
mm.list_models()                      # 已缓存的模型
mm.remove("qwen3")
```
