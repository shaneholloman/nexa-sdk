将 GenieX SDK 接入到 Android Studio 项目中，让你的应用可以从 Hugging Face / Qualcomm AI Hub 拉取权重，并在 Hexagon NPU、Adreno GPU 或 CPU 计算单元上运行它们——全程使用 Kotlin。

## **先体验示例应用**

在编写任何代码之前，先看看 GenieX 在 Android 上的运行效果。

<Card title="Download geniex-demo.apk" icon="download" href="https://qaihub-public-assets.s3.us-west-2.amazonaws.com/qai-hub-geniex/geniex-demo.apk">
  安装到你的手机上，启动 **GenieX Demo**，然后 **Download → Load → Chat**。
</Card>

安装方法：在 **Files**（文件）应用中点击该 `.apk`，如有提示请允许 *安装未知应用*。

从下拉菜单中选择一个模型，并在加载时选择 NPU、GPU 或 CPU。对于 VLM，点击图片按钮。首次下载请保持 Wi-Fi 连接。

源代码位于 [`examples/android/`](https://github.com/qcom-ai-hub/geniex/tree/main/examples/android)——在 Android Studio 中打开它并点击 **Run ▶**。没有手机？请参阅[在没有真机的情况下测试](/cn/resources/faq#how-do-i-test-the-android-demo-without-a-physical-device)。

## **前置条件**

- **Android Studio** [Hedgehog (2023.1.1)](https://developer.android.com/studio/releases/past-releases/as-hedgehog-release-notes) 或更新版本。
- 应用模块中 **`minSdk = 27`**（Android 8.1）。
- 一台搭载 **骁龙 8 至尊版**（`SM8750`）或 **骁龙 8 至尊版 Gen 5**（`SM8850`）的手机——参阅[支持的平台](/cn/get-started/platforms)。

## **将 SDK 添加到你的应用**

<Steps>
  <Step title="Enable Maven Central">
    在 `settings.gradle.kts`（或较旧项目的顶层 `build.gradle.kts`）中：

    ```kotlin
    dependencyResolutionManagement {
        repositories {
            google()
            mavenCentral()
        }
    }
    ```
  </Step>

  <Step title="Add the dependency">
    在你的 **应用模块** 的 `build.gradle.kts` 中：

    ```kotlin
    dependencies {
        implementation("com.qualcomm.qti:geniex-android:0.3.1")
    }
    ```

    该构件附带原生 `arm64-v8a` 库——你这边无需 NDK 或 CMake。
  </Step>

  <Step title="Declare permissions">
    SDK 会在运行时拉取权重。在 `AndroidManifest.xml` 中：

    ```xml
    <uses-permission android:name="android.permission.INTERNET" />
    ```

    对于从相册加载图片的 VLM，还需声明 `READ_EXTERNAL_STORAGE`（或 Android 13+ 上的范围化媒体权限）。
  </Step>

  <Step title="Sync Gradle">
    在 Android Studio 中点击 **Sync Now**。
  </Step>
</Steps>

接下来，前往[快速入门](/cn/run/android/quickstart)下载并运行你的第一个模型。
