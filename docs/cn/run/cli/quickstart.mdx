## **前置条件**

- 已安装 CLI——详见[安装](/cn/run/cli/install)。
- 容器内交互式 shell（仅 Docker）——详见[交互式运行](/cn/run/cli/install#交互式运行)。
- 了解[运行时选择](/cn/get-started/choose-runtime)——`qairt` 用于 AI Hub 模型包，`llama_cpp` 用于任意 GGUF。

## **运行你的第一个模型**

### **QAIRT 运行时（AI Hub）**

**语言模型：**

```powershell windows
geniex infer ai-hub-models/Qwen3-4B
```

**多模态模型：**

```powershell windows
geniex infer ai-hub-models/Qwen2.5-VL-7B-Instruct
```

### **llama.cpp 运行时（GGUF）**

系统提示量化时请选择 `Q4_0`——它在 Hexagon NPU 上支持最佳。

**语言模型：**

```powershell windows
geniex infer unsloth/Qwen3.5-0.8B-GGUF
```

**多模态模型：**

```powershell windows
geniex infer Qwen/Qwen3-VL-2B-Instruct-GGUF
```

系统提示时请选择：
- **模型类型** —— 视觉语言模型选 `vlm`，纯文本模型选 `llm`。`Qwen3.5` 与 `Gemma4` 目前请选 `llm`（多模态支持即将推出）。
- **量化** —— `Q4_0` 在 Hexagon NPU 上性能最佳。

<Tip>如需尝试其他 GGUF 模型，从 Hugging Face 复制任意兼容 GGUF 路径替换上面的命令即可。详见[自带模型](/cn/models/bring-your-own)。</Tip>
