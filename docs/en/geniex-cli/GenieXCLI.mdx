## **Model Inference**

### **geniex pull**

Download a model and store it locally.
After entering the pull command, you will be guided through a setup process to choose the model type, main model file, tokenizer (optional), and extra files (optional).

For `QAIRT` plugin models, no quantization precision selection is needed. For `llama.cpp` plugin models, after running `geniex pull <model-name>`, the CLI will prompt you to select a quantized version if multiple are available:

```bash
Choose a quant version to download
> Q4_0       [1.2 GiB] (default)
  Q8_0       [2.0 GiB]
  F16        [3.8 GiB]
```

### **geniex infer — LLM**

Launch an interactive chat session with a language model.

```bash
geniex infer Qualcomm/Qwen3-4B
```

Use the `--think` / `--no-think` flag to control whether the model outputs its internal reasoning process before the final response:

```bash
geniex infer Qualcomm/Qwen3-4B --think
geniex infer Qualcomm/Qwen3-4B --no-think
```

- `--think` — The model displays its reasoning steps before the final response.
- `--no-think` — The model responds directly without showing reasoning.

### **geniex infer — VLM**

Run vision-language inference with text-only or image input:

```bash
geniex infer Qualcomm/Qwen2.5-VL-7B-Instruct-GGUF
```

If you only want text input, simply launch the command and begin chatting.
To have the model respond from an image, provide the **absolute path** or simply drag the image file into your terminal, for example:

```bash
Describe this picture </full/path/to/image.png>
```


## **Utility Commands**

| Command | Description | Example |
|---|---|---|
| `geniex list` | Display all downloaded models with their names and sizes. | `geniex list` |
| `geniex remove <model>` | Remove a specific local model by name. | `geniex remove unsloth/Qwen3-0.6B-GGUF` |
| `geniex clean` | Delete all locally cached models. | `geniex clean` |
| `geniex infer -h` | Show help menu for `geniex infer`. | `geniex infer -h` |
| `geniex serve -h` | Show help menu for `geniex serve`. | `geniex serve -h` |
