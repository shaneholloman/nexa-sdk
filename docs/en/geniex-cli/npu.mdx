## **Supported Model List**
We curated a list of top, high quality models in QAIRT `bin` format.

| Model ID | Resources |
|---|---|
| qualcomm/Qwen3-4B | Qualcomm AI Hub |
| qualcomm/Qwen3-4B-Instruct-2507 | Qualcomm AI Hub |
| qualcomm/Qwen2.5-VL-7B-Instruct | Qualcomm AI Hub |
| llama_v3_1_8b_instruct | Self converted and uploaded to personal Hugging Face |
| llama_v3_1_sea_lion_3_5_8b_r | Self converted and uploaded to personal Hugging Face |
| llama_v3_2_1b_instruct | Self converted and uploaded to personal Hugging Face |
| llama_v3_2_3b_instruct | Self converted and uploaded to personal Hugging Face |

| falcon_v3_7b_instruct | Self converted and uploaded to personal Hugging Face |

To run a model on `qualcomm` Huggingface, take `Qwen3-4B` for example, simply run:

```powershell
geniex infer qualcomm/Qwen3-4B
```

For models we self converted and uploaded to personal Hugging Face , take `llama_v3_2_1b_instruct` for example, you need to firstly download the HF weights locally then run with the local path. 

Follow this [Huggingface CLI download tutorial](https://huggingface.co/docs/huggingface_hub/en/guides/cli) to setup HF CLI first, then in your windows laptop, run below: 

```powershell
# Set model name (must match the HF subfolder name exactly)
$model = "llama_v3_2_1b_instruct"

# Download the model from Huggingface
hf download yichqian/geniex-qairt-models `
  --local-dir $model `
  --include "$model/**"

# Pull using the downloaded folder (absolute path)
geniex pull local/$model --model-hub localfs `
  --local-path (Resolve-Path "$model\$model").Path

# Run inference
geniex infer local/$model
```
