## **Installation & Running Your First Model**

<Tabs>
  <Tab title="Windows ARM64">
    ### **Windows ARM64 Installation**

    Download the installer:

    - [GenieX CLI](https://github.com/qcom-ai-hub/geniex/actions/runs/25328234046/artifacts/6787639374)

    Run the downloaded `.exe` file and follow the installation.

    ### **Running Your First Model through QAIRT plugin**

    Currently, we support LLM and Multimodal models. More model type support is coming soon!

    **Language Model (LLM)**

    ```bash Windows
   geniex infer qualcomm/Qwen3-4B
   geniex infer qualcomm/Qwen3-4B-Instruct-2507
    ```

    **Multimodal Model**
    ```bash Windows
    geniex infer qualcomm/Qwen2.5-VL-7B-Instruct
    ```

    ### **Running Your First Model through llama.cpp plugin**
    **Language Model (LLM)**

    ```bash Windows
   geniex infer unsloth/Qwen3.5-0.8B-GGUF
   geniex infer unsloth/Phi-4-mini-instruct-GGUF
    ```

    **Multimodal Model**
    ```bash Windows
    geniex infer Qwen/Qwen3-VL-2B-Instruct-GGUF
    ```

    <Tip>To try other GGUF models, visit Hugging Face, copy the path of any compatible GGUF model (e.g., `unsloth/Qwen2.5-VL-3B-Instruct-GGUF`), and replace the model path in the command above.</Tip>

  </Tab>

</Tabs>
