# GenieX Python SDK API Reference

The GenieX Python SDK follows the same design patterns as Hugging Face `transformers` â€” use `AutoModel*.from_pretrained()` to load a model, then call `.generate()` for inference.

## AutoModelForCausalLM

Factory for loading text-only causal language models. Returns a [`GeniexLLM`](#geniexllm) instance.

```python
from geniex import AutoModelForCausalLM

model = AutoModelForCausalLM.from_pretrained(
    "Qualcomm/Qwen3-4B-Instruct",   # HuggingFace repo id, short alias, or local path
    device_map="auto",               # "auto" | "<plugin>" | "<plugin>:<device>"
)
```

### `from_pretrained()`

| Parameter | Type | Default | Description |
|---|---|---|---|
| `model_name_or_path` | `str` | *required* | HuggingFace repo id (e.g. `"Qualcomm/Qwen3-4B-Instruct"`), short alias (e.g. `"qwen3"`), or local path. |
| `model_name` | `str \| None` | `None` | Override the registry model name (e.g. `"granite4"` for QAIRT). Defaults to `model_name_or_path`. |
| `quant` | `str \| None` | `None` | Quantization variant (e.g. `"Q4_K_M"`). Used to filter files when downloading from HuggingFace Hub. |
| `device_map` | `str` | `"auto"` | `"auto"` picks the first available plugin/device. Also accepts `"<plugin_id>"` or `"<plugin_id>:<device_id>"`. |
| `n_ctx` | `int` | `0` | Context length (`0` = model default). |
| `n_gpu_layers` | `int` | `-1` | Layers to offload to GPU (`-1` = all). |
| `tokenizer_path` | `str \| None` | `None` | Override tokenizer file path. |
| `license_id` | `str \| None` | `None` | NPU licence ID. |
| `license_key` | `str \| None` | `None` | NPU licence key. |
| `hf_token` | `str \| None` | `None` | HuggingFace bearer token for gated models. |
| `enable_thinking` | `bool` | `False` | Enable thinking mode (e.g. Qwen reasoning models). |
| `n_threads` | `int` | â€” | Number of threads for inference. |
| `n_batch` | `int` | â€” | Batch size for prompt processing. |
| `max_tokens` | `int` | â€” | Maximum tokens for generation. |
| `system_prompt` | `str` | â€” | System prompt to prepend. |

**Returns:** [`GeniexLLM`](#geniexllm)

## AutoModelForVision2Seq

Factory for loading vision-language / multimodal models. Returns a [`GeniexVLM`](#geniexvlm) instance.

```python
from geniex import AutoModelForVision2Seq

model = AutoModelForVision2Seq.from_pretrained("qwen3vl", device_map="auto")
```

### `from_pretrained()`

Accepts all the same parameters as [`AutoModelForCausalLM.from_pretrained()`](#from_pretrained) plus:

| Parameter | Type | Default | Description |
|---|---|---|---|
| `mmproj_path` | `str \| None` | `None` | Path to the multimodal projector file. Auto-resolved when downloading from Hub. |

**Returns:** [`GeniexVLM`](#geniexvlm)

## GeniexLLM

Text-only language model instance returned by `AutoModelForCausalLM.from_pretrained()`.

### `generate()`

Run text generation from a formatted prompt string.

```python
output = model.generate(prompt, max_new_tokens=256)
print(output.text)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `prompt` | `str` | *required* | The formatted prompt string (use `model.tokenizer.apply_chat_template()` to build it). |
| `max_new_tokens` | `int` | `512` | Maximum number of tokens to generate. |
| `temperature` | `float` | `0.7` | Sampling temperature. |
| `top_p` | `float` | `0.9` | Nucleus sampling threshold. |
| `top_k` | `int` | `40` | Top-k sampling. |
| `min_p` | `float` | `0.0` | Minimum probability threshold. |
| `repetition_penalty` | `float` | `1.1` | Repetition penalty. |
| `presence_penalty` | `float` | `0.0` | Presence penalty. |
| `frequency_penalty` | `float` | `0.0` | Frequency penalty. |
| `seed` | `int` | `-1` | Random seed (`-1` = random). |
| `stop` | `list[str] \| None` | `None` | Stop sequences. |
| `grammar` | `str \| None` | `None` | GBNF grammar string for constrained generation. |
| `json_mode` | `bool` | `False` | Force JSON output. |
| `stream` | `bool` | `False` | If `True`, returns a [`TextIteratorStreamer`](#textiteratorstreamer) instead. |

**Returns:** [`GenerateOutput`](#generateoutput) (or [`TextIteratorStreamer`](#textiteratorstreamer) when `stream=True`)

### `reset()`

Resets conversation state and clears the KV cache.

### `save_kv_cache(path)`

Saves the current key-value cache to a file.

| Parameter | Type | Description |
|---|---|---|
| `path` | `str` | File path to save the cache. |

### `load_kv_cache(path)`

Loads a previously saved key-value cache from a file.

| Parameter | Type | Description |
|---|---|---|
| `path` | `str` | File path to load the cache from. |

### `close()`

Releases the model handle and frees resources. Also supports context-manager usage:

```python
with AutoModelForCausalLM.from_pretrained("qwen3") as model:
    output = model.generate(prompt)
```

## GeniexVLM

Vision-language model instance returned by `AutoModelForVision2Seq.from_pretrained()`.

### `generate()`

Same parameters as [`GeniexLLM.generate()`](#generate) plus:

| Parameter | Type | Default | Description |
|---|---|---|---|
| `images` | `list[str] \| None` | `None` | List of image file paths for the model to process. |
| `audios` | `list[str] \| None` | `None` | List of audio file paths for the model to process. |

```python
output = model.generate(prompt, images=["/path/to/image.jpg"], max_new_tokens=256)
print(output.text)
```

**Returns:** [`GenerateOutput`](#generateoutput) (or [`TextIteratorStreamer`](#textiteratorstreamer) when `stream=True`)

### `reset()`

Resets conversation state and clears the KV cache.

### `close()`

Releases the model handle and frees resources. Supports context-manager usage.

## ModelTokenizer

Accessed via `model.tokenizer`. Provides a `transformers`-compatible chat template interface.

### `apply_chat_template()`

Formats a list of chat messages using the model's built-in chat template.

```python
messages = [{"role": "user", "content": "What is 2+2?"}]
prompt = model.tokenizer.apply_chat_template(
    messages, tokenize=False, add_generation_prompt=True,
)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `messages` | `list[dict]` | *required* | List of message dicts with `"role"` and `"content"` keys. |
| `tokenize` | `bool` | `False` | Must be `False` â€” standalone tokenization is not supported. |
| `add_generation_prompt` | `bool` | `True` | Whether to append the generation prompt suffix. |
| `enable_thinking` | `bool` | `False` | Enable thinking mode (e.g. Qwen reasoning models). |
| `tools` | `list[dict] \| str \| None` | `None` | Tool definitions as a list of dicts or pre-serialised JSON string. |

**Returns:** `str` â€” formatted prompt ready for `model.generate()`.

## Output Classes

### GenerateOutput

Returned by `model.generate()`.

| Attribute | Type | Description |
|---|---|---|
| `text` | `str` | The generated text (with thinking tags stripped if present). |
| `thinking` | `str \| None` | The model's reasoning content (extracted from `<think>` tags), or `None`. |
| `profile` | [`ProfileData`](#profiledata) | Performance metrics. |

### ProfileData

Performance profiling data for a generation call.

| Attribute | Type | Description |
|---|---|---|
| `ttft` | `int` | Time to first token (ms). |
| `prompt_time` | `int` | Prompt processing time (ms). |
| `decode_time` | `int` | Decode time (ms). |
| `prompt_tokens` | `int` | Number of prompt tokens. |
| `generated_tokens` | `int` | Number of generated tokens. |
| `prefill_speed` | `float` | Prefill speed (tokens/s). |
| `decode_speed` | `float` | Decode speed (tokens/s). |
| `stop_reason` | `str \| None` | Why generation stopped (e.g. `"eos"`, `"limit"`). |

### TextIteratorStreamer

Returned by `model.generate(..., stream=True)`. Yields decoded text chunks as they are generated.

```python
streamer = model.generate(prompt, max_new_tokens=256, stream=True)
for chunk in streamer:
    print(chunk, end="", flush=True)

# After iteration, the final output is available:
final = streamer.output  # GenerateOutput
```

| Method / Property | Description |
|---|---|
| `__iter__()` | Yields `str` chunks as they are generated. |
| `output` | `GenerateOutput \| None` â€” the complete result, available after iteration finishes. |
| `cancel()` | Stop generation at the next token boundary. |

## Model Manager

The same model manager the CLI uses is available programmatically via `geniex.model_manager`.

```python
from geniex import model_manager as mm

mm.pull("qwen3")                       # download by alias or "org/repo[:quant]"
models = mm.list_models()              # list cached model names
paths  = mm.get_paths("qwen3")        # resolve to local file paths
mtype  = mm.get_type("qwen3")         # "llm" or "vlm"
name   = mm.resolve_alias("qwen3")    # "Qualcomm/Qwen3-4B-Instruct"
mm.remove("qwen3")                    # delete one cached model
mm.clean()                            # delete all cached models
```

### `pull()`

| Parameter | Type | Default | Description |
|---|---|---|---|
| `model_name` | `str` | *required* | `org/repo` or a short alias. |
| `quant` | `str \| None` | `None` | Quantization hint (e.g. `"Q4_K_M"`). |
| `hub` | `str` | `"auto"` | `"auto"` \| `"hf"` \| `"localfs"`. |
| `local_path` | `str \| None` | `None` | Source directory (required when `hub="localfs"`). |
| `hf_token` | `str \| None` | `None` | HuggingFace bearer token. |
| `on_progress` | `Callable \| None` | `None` | Callback `(files: list[FileProgress]) -> bool`; return `False` to cancel. |

### `list_models()` â†’ `list[str]`

Returns cached model names.

### `get_paths(model_name)` â†’ `ModelPaths`

Resolves a model name to absolute local paths.

| Attribute | Type | Description |
|---|---|---|
| `model_path` | `str` | Path to the main model file. |
| `model_dir` | `str` | Directory containing the model. |
| `model_name` | `str` | Canonical model name. |
| `plugin_id` | `str` | Plugin identifier. |
| `mmproj_path` | `str \| None` | Multimodal projector path (VLM only). |
| `tokenizer_path` | `str \| None` | Tokenizer file path. |
| `device_id` | `str \| None` | Device identifier. |

### `get_type(model_name)` â†’ `str`

Returns `"llm"` or `"vlm"`.

### `resolve_alias(alias)` â†’ `str`

Resolves a short alias (e.g. `"qwen3"`) to a canonical `org/repo`.

### `remove(model_name)` â†’ `None`

Deletes a cached model from disk.

### `clean()` â†’ `int`

Removes all cached models. Returns the count removed.

## Runtime Functions

Top-level functions for managing the GenieX runtime.

| Function | Description |
|---|---|
| `geniex.init()` | Initialize the GenieX runtime. Called automatically on first model load. |
| `geniex.deinit()` | Shut down the runtime and release resources. |
| `geniex.version()` | Returns the SDK version string. |
| `geniex.get_plugin_list()` | Returns `list[str]` of available plugin IDs (e.g. `["llama_cpp"]`). |
| `geniex.get_device_list(plugin_id)` | Returns `list[tuple[str, str]]` of `(device_id, device_name)` pairs for a plugin. |
