## **GenieX Python SDK**

The GenieX AI provides a comprehensive Python API for on-device AI inference across Large Language Models (LLM) and Vision-Language Models (VLM). It is designed to work similar to Huggingface `transformers` library to provide a familiar and intuitive user exprience for developers.
