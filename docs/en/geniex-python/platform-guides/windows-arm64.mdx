## **Requirements**

- Windows 11 ARM64
- Python 3.11–3.13
- Snapdragon X Elite device

## **Install**

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install nexaai
```

## **Quick Test**

```python
from nexaai import VLM, ModelConfig
vlm = VLM.from_(model="NexaAI/OmniNeural-4B", config=ModelConfig(), plugin_id="npu")
print("OK")
```
