## **Quick Overview**

### **Method 1 : Use Python script**

For QDC, we recommend using the Python script method below.

### **Installation**

On QDC, download [Python 3.13.3 for ARM64](https://www.python.org/ftp/python/3.13.3/python-3.13.3-arm64.exe) and then install the GenieX Python SDK using pip:

```bash
pip install -i https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple geniex
```
This will download the python package from [test PyPI](https://test.pypi.org/project/geniex/)

<Note>
**Troubleshooting: SSL / certificate errors (corporate proxy / QDC)**

If the installation fails with an SSL certificate error — common on corporate networks with TLS inspection, including **Qualcomm Developer Cloud (QDC)** — pre-download the SDK manually and point the installer to it. Run the following **PowerShell** commands **before** `pip install`:

```powershell
$env:GENIEX_VERSION = "0.1.4"  # active version — replace with the version you are installing
$base = "https://qaihub-public-assets.s3.us-west-2.amazonaws.com/qai-hub-geniex/$env:GENIEX_VERSION"
$zip  = "geniex-sdk-windows-arm64-$env:GENIEX_VERSION.zip"

Invoke-WebRequest "$base/$zip"        -OutFile $zip
Invoke-WebRequest "$base/$zip.sha256" -OutFile "$zip.sha256"

$env:GENIEX_SDK_DOWNLOAD_URL = "file:///$($(Get-Location).Path.Replace('\','/'))"
pip install -i https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple geniex

Remove-Item $zip, "$zip.sha256"
Remove-Item Env:GENIEX_SDK_DOWNLOAD_URL
```
</Note>

### **LLM Inference**

Here's a simple example to get you started:

```python
from geniex import AutoModelForCausalLM

# GGUF Model Example
model = AutoModelForCausalLM.from_pretrained(
    "Qwen/Qwen3-0.6B-GGUF",     # HF repo id of a GGUF model, or a local path to a .gguf LLM model
    device_map="auto",          # "auto" (default) | "cpu" | "gpu" | "npu" | "hybrid"
                                # | "<plugin>" | "<plugin>:<device>"
                                # auto -> hybrid for llama_cpp (fast HTP+CPU path), npu for qairt
)

# QAIRT Model Example
# model = AutoModelForCausalLM.from_pretrained( 
#     r"C:\<your_path>\Qwen3-4B-Instruct-2507\",    # replace with your own local path; pass a path to any .bin file from the model folder.
#     model_name="qwen3_4b_instruct_2507",          # model name is the same as model id used on QAI Hub Model; they can be found from the S3 download URL.
#     device_map="qairt",                           # device_map must be "qairt" for QAIRT models
# ) 


messages = [{"role": "user", "content": "What is 2+2?"}]
prompt = model.tokenizer.apply_chat_template(
    messages, add_generation_prompt=True,
)

# One-shot
output = model.generate(prompt, max_new_tokens=256)
print(output.text)
print(f"[{output.profile.generated_tokens} tok, "
      f"{output.profile.decode_speed:.1f} tok/s, stop={output.profile.stop_reason}]")

# Streaming
streamer = model.generate(prompt, max_new_tokens=256, stream=True)
for chunk in streamer:
    print(chunk, end="", flush=True)

model.close()
```

### **VLM Inference**

First, download a sample image:

```bash
curl -o demo.jpg https://qaihub-public-assets.s3.us-west-2.amazonaws.com/qai-hub-geniex/demo.jpg
```

Then run inference using the full path to the downloaded image:

```python
import os
from geniex import AutoModelForVision2Seq

image_path = os.path.abspath("demo.jpg")

model = AutoModelForVision2Seq.from_pretrained("unsloth/Qwen2.5-VL-3B-Instruct-GGUF", quant="Q8_0", device_map="auto")
messages = [{
    "role": "user",
    "content": [
        {"type": "image", "image": image_path},
        {"type": "text", "text": "Describe the image."},
    ],
}]
prompt = model.tokenizer.apply_chat_template(
    messages, tokenize=False, add_generation_prompt=True,
)
print(model.generate(prompt, images=[image_path]).text)
```
