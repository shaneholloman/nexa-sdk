GenieX is an on-device AI inference runtime built for Qualcomm Snapdragon platforms. It runs frontier language and vision-language models locally on Hexagon NPU, Adreno GPU, or CPU — with no cloud dependency.

## **What you can do with GenieX**

- **Run models locally** on Snapdragon X (Windows on Arm), Snapdragon 8 Elite (Android), and Dragonwing IoT devices.
- **Pick a backend** — `llama.cpp` for any community GGUF model, `QAIRT` for AI Hub pre-compiled NPU bundles.
- **Build apps** through the CLI, an OpenAI-compatible local server, the Python SDK, the Android SDK, or a Docker image.
